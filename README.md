# teacher-dashboard-grades
"إضافة نظام إدارة العلامات والتحليل الإحصائي"# 📊 نظام إدارة العلامات والتحليل الإحصائي

## 🎯 عن المشروع
نظام متكامل لإدراج علامات التلاميذ وتحليلها إحصائياً بسرعة فائقة، مع رسوم بيانية وتقارير فردية قابلة للطباعة.

## 🛠️ التقنيات المستخدمة
- HTML5
- CSS3 (تصميم متجاوب)
- JavaScript (Vanilla JS)

## ✨ الميزات
- لصق البيانات من Excel بنقرة واحدة
- تحليل إحصائي فوري (المتوسط، الانحراف المعياري، الربيعيات، المنوال)
- رسوم بيانية تفاعلية (الأعمدة، الدائرة، Box Plot)
- تقارير فردية مفصلة لكل تلميذ
- طباعة احترافية للنتائج والتحليلات
